# 🎷 Interactive Saxophone Guide

Um **site interativo** criado para ajudar estudantes de **conservatórios de música** e iniciantes a entender as diferenças entre os principais tipos de saxofone.

O projeto apresenta os quatro saxofones mais utilizados na música e explica suas características, história e aplicações musicais de forma visual e interativa.

---

## 🎯 Objetivo do Projeto

Muitos estudantes que começam a estudar saxofone têm a mesma dúvida:

> **Qual saxofone devo escolher para aprender?**

Este projeto foi criado para responder essa pergunta de forma **simples, visual e interativa**, permitindo que o usuário navegue pelos instrumentos e descubra qual pode se encaixar melhor em seu estilo musical.

---

## 🎷 Saxofones apresentados

### 🎷 Saxofone Soprano
- Som mais **agudo**
- Muito utilizado em **jazz e música erudita**
- Instrumento reto (parecido com um clarinete)

### 🎷 Saxofone Alto
- Um dos **mais populares para iniciantes**
- Muito usado em **bandas, orquestras e jazz**
- Excelente equilíbrio entre grave e agudo

### 🎷 Saxofone Tenor
- Som mais **encorpado e quente**
- Muito presente no **jazz, blues e música popular**
- Instrumento bastante expressivo

### 🎷 Saxofone Barítono
- O **mais grave** entre os quatro
- Muito usado para **base harmônica em bandas**
- Instrumento grande e poderoso

## 🧠 O que o usuário encontra no site

O usuário pode navegar pelo projeto e visualizar:

- História de cada saxofone
- Características sonoras
- Aplicações na música
- Perfil de músicos que costumam escolher o instrumento

Tudo apresentado de forma **interativa e dinâmica**.
